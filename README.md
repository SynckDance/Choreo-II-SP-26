## Deploy to GitHub Pages

1. Create a new GitHub repository
2. Upload all files from this source package
3. Go to **Settings → Pages**
4. Source: **GitHub Actions**
5. Push to `main` branch — the site will auto-deploy!

### Local Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev
```

---

## Install as App (PWA)

**On iPhone/iPad:**
1. Open in Safari
2. Tap Share button
3. Tap "Add to Home Screen"

**On Android:**
1. Open in Chrome
2. Tap menu (3 dots)
3. Tap "Add to Home Screen"
